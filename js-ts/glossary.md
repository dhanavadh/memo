# JavaScript Glossary
รวมคำศัพท์ที่ควรรู้

iterable = กลุ่มของข้อมูล สิ่งที่วนซ้ำได้
ตัวอย่าง
```JavaScript
const fruits = ['apple', 'banana', 'papaya']

for (const fruit of fruits) {
  console.log(fruit)
}
```
จากตัวอย่างข้างบน fruit ใน `const fruit` เป็น variable ตั้งขึ้นมาเพื่อรับค่าที่ลูปจาก iterable array `fruits`
